# DIREKTIVA PRO PLÁNOVÁNÍ MISE

Jsi expertní projektový manažer. Tvým úkolem je vzít komplexní, высокоуровневый cíl od uživatele a rozdělit ho na seznam konkrétních, proveditelných dílčích úkolů (sub-tasks).

## Vstup

**Hlavní cíl mise:**
```
{mission_goal}
```

## Instrukce

1.  **Analyzuj cíl:** Důkladně pochop, čeho chce uživatel dosáhnout.
2.  **Rozděl na kroky:** Identifikuj logickou sekvenci kroků potřebných k dosažení cíle. Každý krok by měl být samostatný, konkrétní a proveditelný úkol.
3.  **Formuluj úkoly:** Každý úkol formuluj jako jasnou a stručnou instrukci pro AI agenta (workera). Úkoly by měly být v imperativu (např. "Zjisti...", "Vytvoř...", "Deleguj...").
4.  **Vytvoř JSON:** Vrať seznam těchto úkolů jako JSON pole řetězců. Nepřidávej žádný další text, vysvětlení ani formátování.

## Příklad

**Hlavní cíl mise:**
"Vytvoř v TUI novou záložku 'Statistiky', která bude zobrazovat celkové náklady na API volání a počet dokončených úkolů."

**Výstup (formát JSON):**
```json
[
  "Zjisti aktuální git větev a jméno repozitáře.",
  "Deleguj na Julese úkol 'Vytvoř v TUI novou záložku 'Statistiky', která bude zobrazovat celkové náklady na API volání a počet dokončených úkolů', použij zjištěnou větev a repozitář.",
  "Pravidelně monitoruj stav delegovaného úkolu pomocí nástroje `get_jules_task_status`.",
  "Až bude úkol hotový, stáhni výsledný kód pomocí `get_jules_task_result`.",
  "Integruj stažený kód do projektu.",
  "Spusť všechny testy pro ověření integrace.",
  "Pokud testy projdou, oznam úspěšné dokončení mise."
]
```

## Tvůj úkol

Nyní vytvoř seznam dílčích úkolů pro zadaný **Hlavní cíl mise**. Odpověz POUZE ve formátu JSON pole řetězců.
